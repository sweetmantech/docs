Retrieve payment records ‚Äî moments collected through a paid transaction. This is equivalent to calling `GET /transfers` with `type=payment`.

All parameters below apply. The `type` parameter is pre-set to `payment` on this page; to query other transfer types see the related pages:

- [Get Airdrops](/api-reference/airdrops) ‚Äî free distributions (`type=airdrop`)
- [Get Transfers](/api-reference/transfers) ‚Äî all transfer types (no `type` filter)
