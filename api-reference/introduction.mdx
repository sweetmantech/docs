## Base URL

All API requests should be made to:

```
https://recoup-api.vercel.app
```

## Authentication

All API endpoints are authenticated using API keys passed in the `x-api-key` header.

```json
"security": [
  {
    "apiKeyAuth": []
  }
]
```

### Example Request

```bash
curl -X POST https://recoup-api.vercel.app/api/artist \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-api-key" \
  -d '{"name": "Artist Name", "account_id": "your-account-uuid"}'
```
