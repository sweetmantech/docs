---
title: Get Airdrops
openapi: GET /transfers?type=airdrop
---

Retrieve airdrop records — free moment distributions sent directly by an artist to collectors. This is equivalent to calling `GET /transfers` with `type=airdrop`.

All parameters below apply. The `type` parameter is pre-set to `airdrop` on this page; to query other transfer types see the related pages:

- [Get Payments](/api-reference/payments) — paid collections (`type=payment`)
- [Get Transfers](/api-reference/transfers) — all transfer types (no `type` filter)
